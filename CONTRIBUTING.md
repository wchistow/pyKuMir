# Руководство для разработчиков

## Установка зависимостей:

```sh
uv sync
```

## Запуск тестов:

```sh
pytest tests/
```

## Структура тестов:

```
tests/
    test_parser/
        *тесты парсера*
    test_vm/
        mocks/
            *заглушки*
        *тесты виртуальной машины*
```

## Сборка .deb пакета:

```sh
dpkg-deb -b ./debian/ pykumir.deb
```
