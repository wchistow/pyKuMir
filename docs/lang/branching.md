<h1>Команды ветвления</h1>

<h2>Если-иначе</h2>

```kumir
если <выражение> то
  <команды_1>
[иначе
  <команды_2>]
все
```

<i>Выражение может иметь только тип <code>лог</code>.</i>

<h2>Выбор-при</h2>

```kumir
выбор
  при <выражение_1>: <команды_1>
  [при <выражение_2>: <команды_2>
  ...]
  [иначе: <команды_n>]
все
```

<i>Выражение может иметь только тип <code>лог</code>.</i>
