# Команды ветвления

---

## Если-иначе

```1c
если <выражение> то
    <команды 1>
[иначе
    <команды 2>]
все
```

*Выражение может иметь только тип `лог`.*

## Выбор-при

```1c
выбор
    при <выражение 1>: <команды 1>
    [при <выражение 2>: <команды 2>
    ...]
    [иначе: <команды n>]
все
```

*Выражение может иметь только тип `лог`.*
