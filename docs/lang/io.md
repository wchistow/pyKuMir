# Ввод/вывод

## Ввод

```1c
цел а
ввод а
```

---

```1c
вещ а, б
ввод а, б
```

---

```1c
лит имя
вывод "Ваше имя: "
ввод имя
вывод "Привет, ", имя, "!", нс
```

## Вывод

```1c
вывод "привет, мир!"
```

Результат:

```
привет, мир!
```

---

```1c
вывод "2 + 2 будет ", 2 + 2
```

Результат:

```
2 + 2 будет 4
```

---

```1c
цел а := 2 + 3
вывод "2 + 3 будет ", а
```

Результат:

```
2 + 3 будет 5
```

---

```1c
вывод "первая строка", нс, "вторая строка"
```

Результат:

```
первая строка
вторая строка
```
