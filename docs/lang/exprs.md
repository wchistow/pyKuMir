<h1>Вычисления</h1>

<h2>Арифметические операции:</h2>

<ul>
 <li> <code>+</code></li>
 <li> <code>-</code></li>
 <li> <code>*</code></li>
 <li> <code>/</code></li>
 <li> <code>**</code></li>
</ul>

<i>Допускаются операции только над значениями одного типа (<code>5 + 2.5</code> - будет ошибка)</i>

<h2>Логические операции:</h2>

<ul>

 <li> <code><</code></li>
 <li> <code><=</code></li>
 <li> <code>>=</code></li>
 <li> <code>=</code></li>
 <li> <code><></code></li>
 <li> <code>и</code></li>

</ul>
