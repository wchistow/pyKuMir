<h1>Циклы</h1>

<h2>нц N раз</h2>

```kumir
алг нач
  нц 3 + 2 раз
    вывод "тест", нс
  кц
кон
```

Результат:

```
тест
тест
тест
тест
тест
```

<h2>пока</h2>

```kumir
алг нач
  цел а := 5
  нц пока а > 0
    вывод а, нс
    а := а - 1
  кц
кон
```

Результат:

```
5
4
3
2
1
```

<h2>для</h2>

```kumir
алг нач
  нц для а от 0 до 5
    вывод а, нс
  кц
кон
```

Результат:

```
0
1
2
3
4
5
```

<hr/>

```kumir
алг нач
  нц для а от 0 до 5 шаг 2
    вывод а, нс
  кц
кон
```

Результат:

```
0
2
4
```

<h2>до тех пор</h2>

```kumir
алг нач
  цел а := 0
  нц
    вывод а, нс
    а := а + 1
  кц при а < 2
кон
```

Результат:

```
0
1
```

<h2>нц-кц</h2>

<i>Бесконечный цикл</i>

```kumir
алг нач
  нц
    вывод 1, нс
  кц
кон
```

Результат:

```
1
1
1
1
... и т. д.
```
